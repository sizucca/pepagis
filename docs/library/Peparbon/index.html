Peparbon is Pepagis mixin library for Sass.
Source: https://github.com/thoughtbot/bourbon

ライブラリ構成

_core/
├── peparbon/
│   ├── functions/
│   │   ├── _pe-color.scss
│   │   └── ...
│   ├── helpers/
│   │   └── ...
│   ├── css3/
│   │   ├── _pe-animation.scss
│   │   └── ...
│   │── addons/
│   │   ├── _pe-clearfix.scss
│   │   └──  ...
│   │─ _variables.scss
│   └─ _peparbon.scss
└── _core.scss

peparbon/variables

  • グローバル変数の定義。

peparbon/functions/

  • 便利機能の @function 定義ファイル置き場。

peparbon/helpers/

  • ライブラリ内でのみ使用する事を想定とした Helper 関数置き場。

peparbon/css3/

  • CSS 3（ベンダープレフィックス対応）の @mixin 集。

peparbon/addons/

  • CSS 3（ベンダープレフィックス対応）の @mixin 以外の @mixin 定義ファイル置き場。
